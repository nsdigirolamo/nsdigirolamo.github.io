Good Projects

Projects I'm proud to show to you.
Nick's Ray Tracer

A ray tracer written in Rust following the Ray Tracing in One Weekend book series. This is a more serious follow-up to my previous attempt where I did the same exact thing but it was written in C instead. Below are some sample images I made using the ray tracer.

PowerPortals

A lightweight Spigot plugin for Minecraft that adds interconnected portals. This plugin is heavily inspired by the now outdated Wormhole-X-Treme Bukkit plugin. Players can build portals in-game and teleport between them. Each portal has its own unique name, determined by the player that built it. See the wiki page on github for more information.

Unit Circle Toy

A single-file unit circle toy written in python with pygame. Helps to visualize the relationship between a unit circle and different trigonometric functions. Not a lot going on here but I'm still proud of it.

Less Good Projects

Projects that are a little messier than above.
Simple Shell Assignment

A shell written in C for the CISC361: Operating Systems course at the University of Delaware. This was done during the Spring 2023 semester. It has a few built-in shell commands like list, kill, printenv, etc. and has limited support for pipes and file redirects.

Realtime Software Renderer

A realtime software renderer written in Rust. It's incomplete and really untested, but it (barely) works. Also, I managed to import a wireframe of the Utah teapot and recorded a video of me floating around it that you can watch below.
